# LinkedIn Post 1 (Monday) — Idea Inbox

**Voice:** Senior AI strategist. B2B thought leadership. Sharp observation from AI transformation work. No mention of Sama or any product.

**How this file works:**
- Drop ideas in any format — a hook line, a paragraph, a half-formed thought, a transcribed voice memo, a link with a one-line note. The agent will work with whatever you put down.
- Every idea is a `## ` section with a `Status:` line.
- Newest ideas at the top. The agent picks the top-most `Status: unused` idea each Monday.
- After use, the agent flips `Status: unused` → `Status: used YYYY-MM-DD (Week of YYYY-MM-DD)` and adds a one-line note about how it was framed. Ideas stay in the file forever as an audit trail.
- If no `unused` ideas exist, the agent generates its own and notes that in the weekly file.
- **Images (optional):** Drop images in `assets/linkedin/` and reference them with an `Image:` line. The agent will carry the filename through to the weekly content file so you know which image to attach when posting.

**Format example:**

```
## [Short title or hook line]
Status: unused
Added: 2026-05-09
Image: assets/linkedin/filename.jpg — [one-line note on what the image shows]

[The raw idea — a sentence, a paragraph, a link, a question, a half-thought.
Anything you want the agent to work with. No need to polish.]

---
```
